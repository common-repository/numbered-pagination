== Description ==

Adds custom pagination with numbers for posts, using shortcode.

To adjust how it looks, use pagination settings menu.

== Installation ==



1. Upload the plugin files to the `/wp-content/plugins/numbered-pagination` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
